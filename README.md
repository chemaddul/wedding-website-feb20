# Siddhartha & Dhruthi Wedding Website

A beautiful Telugu wedding website for the celebration on February 20, 2025.

## Quick Deploy to Vercel (Easiest)

1. Go to [vercel.com](https://vercel.com) and sign up (free)
2. Click "Add New Project"
3. Import this folder or connect your GitHub repo
4. Click "Deploy" - that's it!

Your site will be live at `your-project-name.vercel.app`

## Quick Deploy to Netlify

1. Go to [netlify.com](https://netlify.com) and sign up (free)
2. Drag and drop this entire folder onto the Netlify dashboard
3. Your site will be live instantly!

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Custom Domain

After deploying, you can add a custom domain like `siddharthadhruthi.com`:

1. Buy a domain from Namecheap, Google Domains, or GoDaddy (~$10-15/year)
2. In Vercel/Netlify settings, add your custom domain
3. Update your domain's DNS settings as instructed
